# SEFAZ XML Downloader

Projeto em Python para consulta automática de XMLs de NF-e via SEFAZ utilizando certificado digital A1.

## Funcionalidades

- Conexão segura com SEFAZ
- Download de XMLs
- Processamento automático
- Logs de execução

## Tecnologias

- Python
- Requests
- SOAP
- Certificado A1

## Como executar

```bash
pip install -r requirements.txt
python app/main.py
